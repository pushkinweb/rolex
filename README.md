# Rolex

## Установка

```bash
# npm install
```

## Режим запуска

```bash
# Режим разработки запуск dev сервера
$ npm run start

# Сборка для продакшена
$ npm run build:prod

# Сборка для разработки
$ npm run build:dev

# Запуск prettier
$ npm run prettier

# Запуск linter для typeScript
$ npm run lint:ts

# Запуск linter для scss
$ npm run lint:scss
```
